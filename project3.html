Template Changes

I used Bootstrap templates to build this portfolio website in order to learn more about manipulating HTML and CSS code to achieve the desired format and content. I started off with the Clean Blog template.

I added new pages, an accordion, and different styles (change in backgrounds, color scheme, footer, social media icons at the bottom, and buttons) to get the end result that you see now. My portfolio website has three main webpages (home, about, and contact) and six project pages. For navigation, there is one menu item for each of the main webpages and a responsive pagination at the bottom of each product page.

While building my website, I also learned:

  • How to inspect code to troubleshoot issues.
  • How to use GitHub to host my website and use Github Desktop to commit or discard changes, keep a log of changes, help me organize my files, and push local commits to origin remote.

I now feel more comfortable using HTML tags and CSS styles.
